# Cálculo de Média Ponderada

## 📚 Descrição

Este programa lê **três valores numéricos** representando as notas de um aluno: **A**, **B** e **C**.

Com base nessas notas, o programa calcula a **média ponderada** do aluno considerando os seguintes pesos:

- Nota **A** → peso **2**
- Nota **B** → peso **3**
- Nota **C** → peso **5**

Cada nota deve estar no intervalo de **0 a 10.0**, sempre com **uma casa decimal**.

## 🧮 Fórmula da Média

A média ponderada é calculada utilizando a seguinte fórmula:

MEDIA = (A × 2 + B × 3 + C × 5) / 10
